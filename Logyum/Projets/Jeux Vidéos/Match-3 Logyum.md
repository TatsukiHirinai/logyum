## Gameplay
Trois mode principal :
- Mode Campagne :
	Une histoire, avec des niveaux variés (faire X score, battre X ennemis, battre le boss en X coups,etc). Ce mode ressemble à One Piece Bonbon Journey.
	Une version hardcore sera disponible.
- Mode Classique : 
	Ce mode est comme le mode campagne mais sans l'histoire et a des niveaux à "l'infini"
- Mode Infini :
	Ce mode est un seul plateau infini, le but est de faire le plus de score possible, il y a un temps qui baisse, et chaque coup / combo augmentent ce temps, le temps diminue de plus en plus vite. Ce mode ressemble à ZooKeeperDX

Tout ces modes utilisent des personnages qui sont obtenables via le mode Campagne ou via un système de Gacha.

Les personnages :
Il y aura toutes les races jouable sauf les Arrint et Aeter. Chaque personnage possède une rareté : Commun, Rare, Légendaire.
